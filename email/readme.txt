1 - O script serve para envio de email utilizando somente ruby e sua biblioteca gem Mail

2 - Ele está bem simples e funciona

3 - O gmail necessita de configuração para aceitar acesso de apps não seguros, o mesmo pode ser visto no link https://support.google.com/a/answer/6260879?hl=pt-BR

4 - para utilizar é necessário a instalação da gem Mail.

     gem install mail

5 - acesse a pasta do arquivo via terminal e execute o comando ruby send.rb e já será possível enviar emails

6 - não esqueça de configurar o servidor no arquivo config.rb

7 - um grande abraço.
